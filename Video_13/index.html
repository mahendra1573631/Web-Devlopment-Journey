Style is 3 types to link

  • External Css
  • Internal Css
  • Inline Css
  1. External Stylesheets (Recommended for most projects):

    2. This method involves creating a separate .css file containing all your styles and linking it to your HTML document.

    • Create a CSS file (e.g., style.css) in the same directory as your HTML file, or in a dedicated css folder.
    • Within the <head> section of your HTML document, add the following <link> tag: 
      • Code:
                        <link rel="stylesheet" href="path/to/your/style.css">
    • rel="stylesheet": Specifies the relationship between the HTML file and the linked file (indicating it's a stylesheet).
    • href="path/to/your/style.css": Specifies the path to your CSS file. Adjust the path if your CSS file is in a different location (e.g., css/style.css if it's in a css folder).

  2. Internal/Embedded Styles (Suitable for single-page documents or page-specific styles) :-

    3. This method involves embedding CSS directly within the <head> section of your HTML document using the <style> tag. 

    Code:
                
                    <head>
                        <style>
                            body{
                                background-color: blue;
                            }
                            h1{
                                color:black;
                            }
                        </style>
                    </head>
  3. Inline Styles (Use sparingly for unique, single-element styling needs):

    4. This method involves applying styles directly to individual HTML elements using the style attribute. 

    Code:
                <h1 style="color:red;font-size: 10px;">
